# Phase 3

This folder contains both Phase 3 Java projects:

- `TicTacToe` - console version
- `TicTacToeGUI` - GUI version

## How to Open the Projects

Open `TicTacToe` or `TicTacToeGUI` directly as the project root in your IDE.

For proper package resolution, do not open `ProjectMaterial/Phase3` or the entire `CS3354_255_DEMO` directory as the project root. Each application should be opened separately because its package structure is defined relative to its own folder.

## How to Run

For the console version:

- Open `ProjectMaterial/Phase3/TicTacToe` as the project root.
- Run `TicTacToe.java`.

For the GUI version:

- Open `ProjectMaterial/Phase3/TicTacToeGUI` as the project root.
- Run `TicTacToe.java`.
